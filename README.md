# ITECT Preview Web

Preview navegable del sitio web informativo para ITECT.

## Páginas incluidas

- Inicio
- Cursos
- Detalle de curso: Desarrollo Python
- Nosotros
- Beneficios
- Contacto
- Inscríbete

## Nota

Esta demo es una preview estratégica para validación comercial. La implementación final del cliente real debe realizarse en WordPress + Elementor con información oficial de cursos, profesores, duración, WhatsApp, correo y certificaciones.
